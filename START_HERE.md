# Start Here

Read this before writing code.

Daedalus is not trying to build better software. Daedalus is trying to help
humans understand reality.

Before changing this repository, read:

1. [How To Think Like Daedalus](ai/HOW_TO_THINK_LIKE_DAEDALUS.md)
2. [Daedalus Platform Constitution v1.2](docs/constitution/DAEDALUS_CONSTITUTION_v1.2.md)
3. [README](README.md)
4. [Daedalus Build Plan Projection](DAEDALUS_BUILD_PLAN.md)

The Constitution tells you what must not be violated. The README and build
plan describe this repo's current scope. The thinking guide gives you the
mental model you need before those documents will make sense.

The short version:

- The Digital Twin is the product.
- Everything else is a projection of the Twin.
- Capture records and reconstructs reality.
- Main explains reality.
- Reports, conversations, AR, accessibility narration, and customer views are
  views of the same truth.
- Evidence attaches to reality.
- Review is reality review, not metadata review.

If a feature does not improve the Twin or improve understanding of the home,
stop and rethink it.
